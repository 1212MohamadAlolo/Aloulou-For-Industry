# تقرير إضافة اللغة الإنجليزية الكاملة — الإصدار v27

## الميزة

تمت إضافة زر لغة واضح داخل الشريط العلوي:

- `EN` للتحويل من العربية إلى الإنجليزية.
- `AR` للعودة من الإنجليزية إلى العربية.

يُحفظ اختيار المستخدم تلقائيًا على الجهاز، لذلك تستمر اللغة المختارة عند:

- الانتقال إلى صفحة تفاصيل المنتج.
- العودة إلى الصفحة الرئيسية.
- فتح سلة المشتريات.
- فتح الموقع كتطبيق PWA.
- إعادة تحميل الموقع.

## ما يتحول إلى الإنجليزية

تمت ترجمة الواجهة كاملة، بما يشمل:

- شريط التنقل.
- الصفحة الرئيسية.
- قسم المنتجات والتصنيفات.
- البحث والفلاتر.
- جميع بطاقات المنتجات الخمس عشرة.
- أسماء المنتجات وأوصافها وحالات التوفر والعلامات التجارية.
- صفحات تفاصيل المنتجات.
- المواصفات الفنية والاستخدامات والتنبيهات.
- سلة المشتريات ونوافذ تحديد الكمية.
- نافذة إتمام الطلب.
- فاتورة الطلب بصيغتي PDF وHTML.
- رسالة الطلب المرسلة عبر WhatsApp.
- زر WhatsApp العائم والرسالة المرافقة له.
- تعليمات تثبيت تطبيق PWA.
- صفحة الكتالوج والطباعة.
- صفحة عدم الاتصال بالإنترنت.
- العناوين والوصف التعريفي وإتاحة الاستخدام.

## اتجاه التصميم

- العربية تعمل باتجاه `RTL`.
- الإنجليزية تعمل باتجاه `LTR`.
- تم ضبط محاذاة النصوص والقائمة والسلة وزر WhatsApp وصفحات التفاصيل.
- تم تحسين حجم زر اللغة وتوزيع عناصر الرأس على شاشات الهاتف.

## إصلاح إضافي مهم

تم توحيد اسم متغير بيانات المنتجات داخل جميع ملفات JavaScript.  
هذا يمنع اختفاء المنتجات أو تعطل فتح تفاصيلها بسبب اختلاف اسم مصدر البيانات بين الملفات.

## الفاتورة وWhatsApp

عند استخدام الموقع بالإنجليزية:

- تُنشأ الفاتورة باللغة الإنجليزية.
- تُكتب أسماء المنتجات والمواصفات والكمية باللغة الإنجليزية.
- تُجهّز رسالة WhatsApp باللغة الإنجليزية.
- يبقى رقم قسم المبيعات:
  `+963 966248480`

وعند العودة إلى العربية تعود الفاتورة والرسالة إلى العربية.

## الكتالوج

تمت إضافة زر لغة داخل صفحة الكتالوج.

في الوضع الإنجليزي:

- تتحول أسماء المنتجات والتصنيفات والعناوين إلى الإنجليزية.
- زر `Download PDF` يفتح نافذة الطباعة لحفظ النسخة الإنجليزية بصيغة PDF.
- لا يتم استخدام ملف PDF العربي القديم عند طلب نسخة إنجليزية.

## PWA والتخزين المؤقت

تم تحديث:

- `service-worker.js`
- اسم الكاش إلى `aloulou-pwa-v27-1`
- إضافة ملف `assets/js/i18n.js` إلى الملفات الأساسية المخزنة.

بعد رفع النسخة إلى GitHub Pages:

1. استبدل جميع ملفات النسخة السابقة.
2. انتظر اكتمال النشر.
3. نفّذ تحديثًا قويًا باستخدام `Ctrl + F5`.
4. إذا كان الموقع مثبتًا كتطبيق، أغلق التطبيق بالكامل ثم افتحه من جديد.
