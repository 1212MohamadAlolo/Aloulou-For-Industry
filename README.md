# علولو للصناعة — النسخة الاستكشافية 1.0

موقع تجريبي ثابت (Static Website) لاستكشاف بنية وهوية موقع منتجات علولو للصناعة.

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح. للحصول على أفضل تجربة شغّل مجلد المشروع عبر خادم محلي، مثل:

```bash
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

## المزايا الحالية

- واجهة عربية RTL متجاوبة.
- وضع فاتح وداكن محفوظ محليًا.
- بحث وفلاتر للمنتجات.
- صفحات تفاصيل ديناميكية عبر `product.html?id=...`.
- مفضلة وطلب عرض سعر باستخدام LocalStorage.
- إنشاء ونسخ رسالة طلب تجريبية.
- صور SVG محلية دون اعتماد على مكتبات صور خارجية.

## تنبيه

جميع المنتجات، الموديلات، المواصفات وبيانات التواصل تجريبية. يجب استبدالها قبل النشر.

## أماكن التعديل الرئيسية

- بيانات المنتجات: `assets/js/products.js`
- الألوان والتصميم: `assets/css/styles.css`
- منطق الصفحة الرئيسية: `assets/js/app.js`
- منطق صفحة المنتج: `assets/js/product.js`
