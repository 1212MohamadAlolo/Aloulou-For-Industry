# تقرير إضافة منتج سيكوينسر المصاعد

## المنتج المضاف
- الاسم: سيكوينسر المصاعد RST
- الاسم الإنجليزي: Elevator RST Signal Sequencer
- العلامة التجارية: علولو للصناعة
- التصنيف: تحكم وأتمتة
- حالة العرض: حسب الطلب

## الوظيفة المعتمدة
- تقليد إشارات R وS وT.
- تمكين تشغيل الإنفرتر عند التغذية من نظام UPS.
- إدخال كرت المصعد ضمن نظام الوصول إلى أقرب طابق أثناء نظام الإنقاذ.
